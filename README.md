# Personal Portfolio Website
This project is a personal portfolio website created using HTML and basic CSS. The website is designed to showcase my profile, skills, and contact information in a clean and simple layout.
# About the Project
This portfolio website includes a home section with my name and role as an Electronics and Communication Engineer (Fresher), along with a contact page that displays my email, phone number, and location. The design is built using basic HTML structure and styled using CSS with colors, spacing, and sections.
This project helped me understand the fundamentals of web development such as HTML tags, CSS styling, page structure, and linking multiple pages.
